# キッザニアで子供が体験する仕事と給料

## 要約
キッザニアでは子供たちが約100種類の職業を体験し、専用通貨「キッゾ」を給料として受け取ります。稼いだキッゾは施設内で使えるほか、銀行に預けて次回に持ち越すことも可能です。実際の現場には派遣されず、施設内のシミュレーションでのみ体験します。

## タグ
Kidsania, 体験, 子供, 経済, シミュレーション

## 内容
- **体験内容**  
  - 約100種類の職業（消防士、医師、パイロット、寿司職人など）を選択  
  - 本格的なユニフォームを着用し、約30分間の作業を体験  
- **給料「キッゾ」**  
  - 仕事を終えるとキッゾが給料として支給  
  - 施設内で買い物や習い事に使用可能  
  - 銀行に預けて次回に持ち越すことも可  
- **実際の現場への派遣はなし**  
  - 施設内に作られた「本物っぽいセット」での体験のみ  
  - 実際の消防署や病院に派遣されることはない  
- **対象年齢**  
  - 3歳くらいから参加できるアクティビティもあり  
  - 9ヶ月の子供にはまだ早いが、成長に合わせて参加可能
